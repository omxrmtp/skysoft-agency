# CRM Components

These components are for the internal CRM dashboard. They use mock data and are not connected to Supabase yet.

## Components

- **CrmShell**: Main CRM layout with sidebar and navigation
- **DashboardView**: Dashboard with statistics and charts (mock data)
- **LeadsView**: Lead management interface (mock data)
- **ProjectsView**: Project management interface (mock data)
